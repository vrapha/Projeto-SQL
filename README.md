# Projeto SQL – Banco de Dados Ecommerce (SQLite)

Este projeto faz parte do meu portfólio de Análise de Dados.  
Aqui eu criei um banco de dados simples para simular um pequeno ecommerce e pratiquei conceitos fundamentais de SQL, como criação de tabelas, inserção de dados e consultas para análise.

---

## Objetivo do Projeto
- Criar um banco de dados SQLite a partir de um script SQL.
- Popular tabelas com dados fictícios (clientes, produtos e pedidos).
- Executar consultas SQL úteis para análise.
- Praticar estruturação de queries e leitura de dados.

---

##  Estrutura do Projeto

project-sql-analysis/
